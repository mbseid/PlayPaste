This project is intended to become a code repository for Play Framework. A place where Play developers can leave code for debugging, or just nice to share. It will be an ongoing project to assemble a large common function library centered around Play Framework web development. Please share some cool tricks you've learned!


Intent


  1. I wanted to learn about the Play Framework and what it can do. By building this side, I learned a whole lot about the framework and how enjoyable it makes java development. Please take the time to review the code, and leave comments on it. This is a learning experience for me, so if you have a better technique, feel free to share.
  2. To be come a code library to share some powerful snippets of code. The tag capability is not included in the application yet, but will be built in shortly. Hopefully, it will become a place of sharing for Play framework related code.

Tech Stack


  1. Play! Framework
  2. PostgresSQL
  3. SCSS
  4. Syntax Highligher
  5. Heroku

Origin


This project was created by Michael Seid, @mbseid. I really liked the looks of Play! Framework, and wanted to try it out. This was a fun learning exercise, and will hopefully will be helpful to the community. Please feel free to send me some comments about my code, or teach me something new!
